# Premium Wording Phase 2

此階段針對其餘前台功能頁的用語進行統一，目標是：

- 減少技術導向或工具式字眼
- 強化內部管理平台的穩重感
- 延續首頁與費用入口的高級文案風格

## 優先替換方向

- 按附件自動讀取並填寫 → 整理附件內容
- 自動讀取內容並填入相關欄位 → 整理附件內容並帶入主要欄位
- 初版會按圖片內容自動讀取英文 / 中文資料 → 系統會按附件內容整理常見資料
- 儲存本地記錄 → 儲存案件記錄
- 本地記錄 → 案件記錄
- 即時核對 → 核對摘要
- 一般模組本地摘要 → 一般模組摘要
