Expense & Reimbursement Portal

費用及報銷入口

現時雜費模組已改為雙入口架構：單次申請 建 1 個 Batch + 1 個 Line；月度總表 建 1 個 Batch + 多個 Line。

01

單次申請

適合一張單據、一筆雜費、一次獨立付款。可上傳收據 / PDF，自動讀取後直接提交。
02

月度總表申請

適合一個月多筆報銷。先填批次資料，再匯入 Excel 建立多條明細。
Data Model

共用儲存結構

  • Batch：主表，存批次資料、店舖、總金額、狀態、月份等。
  • Line：明細表，存每一筆支出日期、費用類型、供應商、金額等。
  • 付款及核對追蹤頁會同時讀取 Batch 與 Line。
Recommended Flow

建議落地方式

  • 前台 HTML：先做原型及本地記錄。
  • Power Automate：之後將 Batch / Line 分開寫入 SharePoint List 或 Excel Table。
  • 追蹤頁：按 BatchNo 展示總額、Line 數量及最後更新時間。
